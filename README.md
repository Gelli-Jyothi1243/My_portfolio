# My Portfolio

A personal portfolio web application built with React, Node.js, Express, and MongoDB.

## 🔗 Live

- **Portfolio**: https://gidy-profile-page.vercel.app
- **GitHub**: https://github.com/Gelli-Jyothi1243/My_portfolio

## 🛠️ Tech Stack

- React + Vite
- Node.js + Express
- MongoDB Atlas
- Deployed on Vercel (frontend) & Render (backend)

## ✨ Features

- Dark / light theme toggle
- Inline editing for all sections
- Skill level badges (Beginner → Expert)
- Education timeline
- Responsive design

## 🚀 Run Locally

**Backend**
```bash
cd server
npm install
# create .env with MONGODB_URI and PORT=5000
npm start
```

**Frontend**
```bash
cd client
npm install
npm run dev
```
